A. Admin Flow
- Login as admin
- Create:
    - a student auth user
    - an instructor auth user
- Create student ERP profile
- Create instructor ERP profile
- Create a course (e.g., CS101)
- Create a section taught by the instructor

B. Student Flow
- Login as student
- Register for the section
- View timetable
- View transcript (empty initially)

C. Instructor Flow
- Login as instructor
- List sections taught
- View enrolled students
- Enter a component grade (e.g., Quiz1 = 85)
- Compute final grade
- Export gradebook

D. Student Final Check
- Login as student
- View grades after instructor graded
- Export transcript
